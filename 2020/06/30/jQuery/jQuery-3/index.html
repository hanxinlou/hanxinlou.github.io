一.获取

1.内容

$(selector).text(); // 返回所选元素的文本内容
$(selector).text(""，callback); // 设置所选元素的文本内容
$(selector).html(); // 返回所选元素的内容
$(selector).html(""，callback); // 设置所选元素的内容
$(selector).val(); // 返回表单字段的值
$(selector).val(""，callback); // 设置表单字段的值

2.属性

$(selector).attr("属性"); // 获取属性值
$(selector).attr("属性", "属性值", callback); // 设置属性值

二. 添加

1.语法

$(selector).append(); // 在被选元素的结尾插入内容
$(selector).prepend(); // 在被选元素的开头插入内容
$(selector).after(); // 在被选元素之后插入内容
$(selector).before(); // 在被选元素之前插入内容

三.删除

1.语法

$(selector).remove(); // 删除被选元素（及其子元素）
$(selector).empty(); // 从被选元素中删除子元素

2.参数

  • 括号中可填选择器来进行过滤

四.操作 CSS

1.语法

$(selector).addClass(); // 向被选元素添加一个或多个类
$(selector).removeClass(); // 从被选元素删除一个或多个类
$(selector).toggleClass(); // 对被选元素进行添加/删除类的切换操作
$(selector).css(); // 设置或返回样式属性

五. css() 方法

1.语法

$(selector).css("propertyname"); // 返回被选元素的一个或多个样式属性
$(selector).css("propertyname","value"); // 设置被选元素的一个或多个样式属性

六. 尺寸方法

1.语法

$(selector).width()
$(selector).height()
$(selector).innerWidth()
$(selector).innerHeight()
$(selector).outerWidth()
$(selector).outerHeight()
