一.语法

1. $(selector).action()

  • 美元符号定义 jQuery
  • 选择符（selector）”查询”和”查找” HTML 元素
  • jQuery 的 action() 执行对元素的操作

二.选择器

1.元素选择器

$("p") :在页面内选取所有 <p> 元素

2. #id 选择器

$("#test") :在页面内选取所有 id="test" 的元素

3. .class 选择器

$(".test") :在页面内选取所有 class="test" 的元素

三.事件

鼠标事件 键盘事件 表单事件 文档/窗口事件
click keypress submit load
dblclick keydown change resize
mouseenter keyup focus scroll
mouseleave blur unload
hover

1. click() 点击事件

  ...

2. dblclick() 双击事件

  ...

3. mouseenter() 鼠标指针穿过元素

4. mouseleave() 鼠标指针离开元素

6. mouseup() 在元素上松开鼠标按钮

7. hover() 光标悬停事件
