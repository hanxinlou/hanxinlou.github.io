Nodejs安装教程

记录一下安装Nodejs中需要注意的问题

1. 下载

官网下载对应系统的安装包

2. 安装

① 运行下载好的安装包一路默认安装即可
② 请注意安装目录最好放在非C盘目录下，且保证路径中无中文或空格
③ 我的安装过程图如下，供参考

④ 最后点击 install 等待安装完毕

3. 环境变量的设置

① 在安装目录下，如 D:\nodejs 新建两个文件夹 node_global(全局包存放目录) 和 node_cache(缓存目录)

② 打开命令行工具，执行以下命令，引号内为安装目录下新建的两个文件夹
npm config set prefix "D:\nodejs\node_global"
npm config set cache "D:\soft\nodejs\node_cache"

③ 环境变量——系统变量中新建变量名：NODE_PATH，变量值：D:\nodejs\node_global\node_modules

④ 编辑用户变量的 Path，将默认的 C 盘下 AppData\Roaming\npm 修改为 D:\nodejs\node_global

⑤ 打开命令行工具测试输入 node -v 和 npm -v 测试是否安装成功

4.关于 npm

  • npm 是 nodejs 的包管理器
  • 用 npm install + 包名 即可下载相应模块包, 模块包的信息（比如版本）会记录在一个名为 package.json 文件里
  • 由于一些原因，使用 npm 安装模块包有时候会遇到下载慢甚至安装失败的情况，
    此时可以使用淘宝镜像 cnpm install + 包名来安装模块包
