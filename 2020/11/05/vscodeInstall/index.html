VSCode 安装及使用说明

一、安装

VSCode 官网下载对应系统版本的安装包，一路默认安装即可

二、使用说明

1. 资源管理器

① 打开文件夹后左侧显示该文件夹下文件目录
② 打开后点击终端则终端目录自动更新为当前目录

③ 终端中创建新文件命令为 code filename.xxx

2. 源代码管理

① 若文件中有git库，对文件进行修改后改动的文件会走左侧更改目录下显示
② 点击该文件会显示文件修改前后对比
③ 点击文件右侧的 + 会将文件暂存

④ 在消息栏里输入原本的 commit 信息点击 ✔ 符号即可提交代码到本地 git 库

⑤ 若本地 git 库关联了远程库，则可以点击 ··· 拉取、推送、同步远程库，其中，左下角的刷新按钮等同于同步操作，左下角也会显示当前分支并且点击可快速切换、创建分支

3. 扩展

① 搜索自己想要的扩展点击 Install 即可下载安装

② 以下为建议安装的扩展

  • Visual Studio Code IntelliCode :根据上下文提供智能的代码完成建议
  • Markdown Preview Enhanced :预览 markdown 文件
  • Color Picker :生成颜色代码
  • Vetur :辅助 Vue 开发

③ 以下为我的扩展安装列表，供参考

通用

  • Visual Studio Code IntelliCode:根据上下文提供智能的代码完成建议
  • Code Runner:运行多种语言的代码段或代码文件
  • Peacock:快速更换你的VS Code的工作区的主题颜色
  • Material Icon Theme:将“材料设计”图标放入VS代码
  • Markdown Preview Enhanced:预览markdown文件
  • Live Share:实时共享代码和协作处理任意内容
  • Color Picker:生成颜色代码

前端

  • Flutter:辅助Flutter开发
  • Dart:支持Dart语言
  • CSS Formatter:格式化CSS文档
  • Vetur:辅助Vue开发

后端

  • Python:辅助Python开发(IntelliSense，linting，调试，代码导航，代码格式，重构，代码段等)
  • Java Test Runner:运行和调试Java测试用例
  • Maven for Java:基于 Maven Archetype 生成 Maven 项目、运行各种 Maven 命令
