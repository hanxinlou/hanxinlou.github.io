HTML

1. HTML是什么

  • HTML不是一种编程语言，而是一种标记语言
  • HTML是一种用标签来描述网页的标记语言
<!-- HTML -->
<html>
<title>我的第一个标题</title>

<body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落</p>
</body>

</html>
  • 最外层的这个html，就代表网页本身
  • body里就是我们平时主要需要编写的内容

2. 最常用的标签

  • div
  • 图片标签（在页面上画图用的，img）
  • 输入框标签（input）
  • 常见的标签和属性表

3. 画一个百度首页

  • 每一个标签需要有自己的属性
    比如src就是img的一个属性，用来存放图片链接
