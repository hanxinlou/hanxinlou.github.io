Vultr+Shadowsocks+Proxy SwitchAyOmega实现自由上网

记录下自己如何实现科学上网

1.购买VPS服务器

  • Vultr
  • Server: Cloud Computer
  • Server Location: Los Angeles
  • Server Type: Ubuntu 16.04 x64
  • Server Size: 55GB SSD $10/month

2.部署VPS服务器

  • 安装Putty 下载地址
  • 输入服务器IP地址、端口(22)、连接类型(SSH)
  • save，load，open
  • 输入用户名(root)、密码(登录Vultr查看服务器密码)
  • 登陆成功
  • 安装wget
  • 安装一键部署SSR脚本(脚本来自于自建ss服务器教程)
  • 管理脚本命令
  • 选择 - 1.安装 ShadowsocksR
  • 设置端口和密码
  • 设置加密方式 - 10.aes-256-cfb
  • 选择协议插件 - 2.auth_shal_v4
  • 选择混淆插件 - 1.plain
  • 部署完成

3.一键加速VPS服务器

  • Google BBR加速
  • 安装成功后reboot重启服务器

4.Shadowsocks设置

  • 安装Shadowsocks
  • 输入服务器地址、端口、密码
  • 选择加密方式 - aes-256-cfb
  • 代理端口 - 1080
  • 右键系统代理 - PAC模式
  • 右键PAC - 使用本地在线PAC
  • 右键开机启动

5.Google浏览器设置

  • 安装扩展程序 Proxy SwitchAyOmega
  • 添加情景模式
  • 输入名称 - SSR
  • 选择类型 - 代理服务器
  • 代理协议 - HTTP
  • 代理服务器 - 127.0.0.1
  • 代理端口 - 1080
  • Google浏览器右上角 SwitchAyOmega - Auto switch
